# Inventory Manager 📦

Eine moderne React-Anwendung zur Verwaltung von Beständen mit **dualen Speichermodi** – ideal für lokale Nutzung oder Integration mit bestehenden APIs.

## Funktionalität

Eine vollständig funktionsfähige Inventarverwaltungs-App mit folgenden Features:

- **Artikel hinzufügen** mit Validierung
- **Artikel bearbeiten** (Name & Menge)
- **Artikel löschen**
- **Bestandsverwaltung** mit Mengentracking
- **Persistierung** – Daten bleiben auch nach Reload erhalten
- **Dual-Mode Support** – localStorage oder REST API
